## Aperçu du projet

Hugo Narrow est un thème Hugo moderne, épuré et minimaliste conçu pour les blogueurs et créateurs de contenu qui valorisent la simplicité et l'élégance. Construit avec Tailwind CSS 4.0, il offre de nombreuses options de personnalisation tout en maintenant d'excellentes performances.

## Fonctionnalités principales

- **Plusieurs schémas de couleurs** : Choisissez parmi plus de 11 thèmes de couleurs pré-conçus
- **Support du mode sombre** : Mode sombre automatique avec transitions fluides
- **Design responsive** : Affichage parfait sur tous les appareils
- **Performances rapides** : Optimisé pour la vitesse et le SEO
- **Support de contenu riche** : Markdown amélioré, coloration syntaxique, équations mathématiques
- **Multilingue** : Support de plus de 13 langues
- **Composants UI modernes** : Recherche, table des matières, commentaires, intégration d'analytics

## Technologies utilisées

Ce projet exploite les technologies web modernes :

- **Hugo Extended** : Générateur de site statique avec templating puissant
- **Tailwind CSS 4.0** : Framework CSS utility-first
- **@tailwindcss/typography** : Beaux paramètres typographiques par défaut
- **GLightbox** : Lightbox élégante pour les images
- **Mermaid** : Support de diagrammes et organigrammes
- **KaTeX** : Composition mathématique rapide

## Philosophie de conception

Le thème suit une philosophie "contenu d'abord", garantissant que votre écriture occupe le devant de la scène. Chaque décision de conception privilégie la lisibilité et l'expérience utilisateur.

### Système de couleurs

Le thème comprend des schémas de couleurs soigneusement élaborés :
- Default, Claude, Bumblebee, Emerald
- Nord, Sunset, Abyss, Dracula
- Amethyst, Slate, Twitter

Chaque thème est conçu pour fonctionner parfaitement en modes clair et sombre.

## Parcours de développement

La construction de Hugo Narrow a été un voyage passionnant d'apprentissage et d'expérimentation. Les principaux défis comprenaient :

1. **Intégration de Tailwind CSS 4.0** : Travailler avec la dernière version nécessitait de s'adapter à une nouvelle syntaxe
2. **Optimisation des performances** : Garantir des temps de chargement rapides tout en maintenant des fonctionnalités riches
3. **Accessibilité** : Rendre le thème utilisable pour tous
4. **Support multilingue** : Mise en œuvre d'une internationalisation complète
